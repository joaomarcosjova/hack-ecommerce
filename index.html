Bem-vindo ao Curso

Olá e seja bem-vindo ao Curso "Programe com o Marcos"!


Você está prestes a iniciar uma jornada empolgante no mundo da programação web. Neste curso, vamos explorar o coração da construção de websites modernos: HTML, CSS e JavaScript. Esses três pilares são essenciais para criar páginas web interativas e atraentes, e nossa missão é guiá-lo em cada passo do caminho para que você se torne um desenvolvedor web habilidoso.

- All images are just used for Preview Purpose Only. They are not part of the template and NOT included in the final purchase files.
- This is not a WordPress theme, It is a HTML5 template

CSS Structure

These are the css files that are loaded into templates in Head Section.

Import google fonts 

								@import url('https://fonts.googleapis.com/css2?family=Poppins:wght@400;500;600;700;900&display=swap');

Import third party CSS library 

								/*import vendors*/
                                @import url(vendors/normalize.css);
                                @import url(vendors/bootstrap.min.css);

                                /*import plugins*/
                                @import url(plugins/jquery-ui.css);
                                @import url(plugins/magnific-popup.css);
                                @import url(plugins/animate.css);

UPDATE

Important Note:

- Backup your website and theme folders completely to make sure you have a restore point in case of any complications.

Update using FTP

  1. Download the latest version of the theme to your device.
  2. Connect to your hosting via FTP software
  3. Upload the theme folder you unzipped to your site folder
  4. You should be asked to confirm overwriting of this folder and its contents. Confirm the overwrite.
